# SQL Sales Analysis Project

##  Objective
The objective of this project is to analyze sales data using SQL and extract meaningful business insights to support decision-making.

---

##  Tools Used
- SQL (PostgreSQL)
- Excel (for dataset)

---

##  Dataset
- The dataset contains 9,994+ sales records including product, region, customer segment, and profit details.

---

##  Key Insights
- Total Orders: 9,994  
- Total Sales: $2,297,200  
- Total Profit: $286,397  

- West region has the highest sales, followed by East  
- Consumer segment contributes the highest revenue  
- Technology category generates the highest profit  

- Some products are making losses, indicating pricing issues  
- Top-selling products include Canon Copier and Fellowes Binding Machine  

- California and New York contribute highest sales  
- New York City has the highest number of orders  

---

##  Project Files
- `queries.sql` → All SQL queries used for analysis  
- `insights.txt` → Final business insights  

---

##  Conclusion
This project demonstrates how SQL can be used to analyze business data, identify trends, and generate actionable insights for better decision-making.
