# Zomato Restaurant Analysis

## Objective
Analyze restaurant data to understand customer preferences and performance.

## Tools Used
- SQL (PostgreSQL)
- Excel

## Dataset
20,000+ records including ratings, cost, location, and cuisines.

## Key Analysis
- Location-wise distribution
- Cost vs Rating
- Online ordering impact
- Cuisine trends

## Insights
- Online ordering restaurants have better ratings  
- Cost does not guarantee quality  
- BTM has highest number of restaurants  
- Quick Bites dominate the market  
- North Indian cuisine is most common  

## Conclusion
Affordable and convenient restaurants perform better.
